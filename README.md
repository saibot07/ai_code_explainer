# AI Code Explainer

AI Code Explainer is a Python-based tool that uses OpenAI's GPT technology to explain code snippets.

## Features
- Input a snippet of code.
- Get detailed, human-readable explanations for the code.
- Supports Python code.

## Requirements
- Python 3.7+
- OpenAI Python library

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ai_code_explainer.git
   cd ai_code_explainer
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Set up OpenAI API credentials in an environment variable `OPENAI_API_KEY`.

## Usage
Run the script:
```bash
python main.py
```
Input your code snippet when prompted, and the tool will generate an explanation.
